An Overview of Lightning Web Components on the SalesforceWay Podcast

I had the honor of being invited on the SalesforceWay Podcast by its host Xi Xiao. We had a great chat during which I presented an overview of Lightning Web Components.

Check out the podcast episode to get the big picture of why Lightning Web Components are a game changer for all developers (not just Salesforce developers).

You'll hear how we're using of the latest and greatest web standards (web components, custom elements, shadow DOM, decorators just to name a few) to let you build blazing fast apps. You'll get an understanding of what's the Lightning Web Component Open Source framework and how it's integrated with the Salesforce Platform.

Last but not least, you'll walk away with some great learning resources such as a curated list of Trailhead badges and a collection of open source sample applications that you can explore.
